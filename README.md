# Wedemy App
This project is for Automated Testing (API and Web) purpose.

## Prerequisites
1. Docker

## Build Docker
1. docker-compose up --no-start
2. docker start w_db w_web (wait about 5 seconds then execute no.3)
3. docker start w_api

## wedemy web url
http://localhost:90

## wedemy api base url
http://localhost:9090/wedemy
